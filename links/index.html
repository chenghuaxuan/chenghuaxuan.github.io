添加友链页面前先执行hexo new links

然后编辑/source/links/index.md在front-matter中添加layout: links

在/source/_data下新建links.yml

添加数据：

- { "name" : "", "link" : "", "description" : "", "avatar" : "" }

name 站点名称(必填)

link 站点地址(必填)

description 站点描述

avatar 站点头像
